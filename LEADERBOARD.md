# Leaderboard — CVSSv5 Skull Emoji

Static. Absurdist. Rigorously peer-reviewed.

This page illustrates how contributions are evaluated.

We do **not** use points.
We use **all four** dimensions.

Small changes can score highly.
A single devastating footnote is respected.
Noise is not rewarded.

---

## 🧭 The Four Dimensions

Each accepted contribution is evaluated across:

| Dimension       | Scale                                        | What It Means                                              |
|-----------------|----------------------------------------------|------------------------------------------------------------|
| **#funny**      | meh / audible laughter / by Turing's beard   | Did it land? Did it make someone snort?                    |
| **#craft**      | loose / solid / surgical                     | How clean and careful was the execution?                   |
| **#coherence**  | random / aligned / disturbingly aligned      | Does it fit the tone and purpose of the document?          |
| **#rfc-energy** | hand-wavy / compliant / worryingly normative | Does it respect the form while gleefully abusing the form? |

This is not a competition.
This is a sorting ritual.

## 📜 What Gets You on the Leaderboard

We look for:

- A joke that is funnier because it is also correct
- Proper (ab)use of MUST, SHOULD, MAY, and the Evil Bit
- RFC archaeology that illuminates something true about standards culture
- Restraint — knowing _precisely_ when to stop
- Understanding the tone

We do not reward:

- Jokes that need explaining
- Dependency creep (normative references must pull their weight)
- Chaos without meaning
- **Anything that would make the document less likely to be accepted by the IETF in 2027**

## 🧠 Teenager Explanation (Plain Version)

If you are a student and reading this:

We are not grading how many jokes you submit.

We are asking:

- Did you make something funnier?
- Did you make something more precise?
- Did you understand the vibe?
- Did you avoid breaking the RFC formatting?

If yes — you're doing it right.
